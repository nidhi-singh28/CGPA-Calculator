# 🎓 CGPA Calculator

A simple Flask-based web app that calculates CGPA based on semester-wise SGPAs entered by the user. Features dynamic input fields and basic validation.

## 🛠️ Technologies Used

- Python  
- Flask  
- HTML/CSS
